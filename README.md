homebrew-tap
============

个人 Homebrew 软件源，包含以下工具：

| 工具 | 描述 |
|------|------|
| [aliyundrive-cli](https://github.com/chyroc/aliyundrive-cli) | 阿里云盘命令行客户端 |
| [hcloud](https://support.huaweicloud.com/productdesc-hcli/hcli_01.html) | 华为云命令行工具 |
| [sshpass](https://sourceforge.net/projects/sshpass/) | SSH 密码自动输入工具 |

## 使用方法

```bash
brew tap timebye/tap
brew install <formula>
```

例如安装 hcloud：

```bash
brew tap timebye/tap
brew install hcloud
```
